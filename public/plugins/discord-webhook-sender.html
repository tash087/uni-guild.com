SYSTEM / DISCORD

DiscordWebhookSender

サーバーの「今」を、Discordへダイレクトに届ける。

プラグイン概要

サーバー内の出来事をリアルタイムで Discord に通知するためのプラグインです。 外部ライブラリ OkHttp をプラグイン内に同梱（Shade）してビルドすることで、複雑な依存関係の設定なしに Java から直接 Discord API へデータを送信します。

技術仕様: 外部APIとの通信に OkHttp を使用。非同期処理により、メインスレッド（サーバーの動作）を妨げることなく通知を送信します。

主な機能

  • メッセージ送信: 指定した Webhook URL を通じて、Discord チャンネルへメッセージを投稿します。
  • コマンド連携: ゲーム内から /dw <メッセージ> を実行することで、即座に Discord へ転送可能です。
  • 柔軟なカスタマイズ: config.yml から Webhook URL や Bot名、アイコンなどを自由に変更できます。

設定例 (Configuration)

# Discord Webhook URL
webhook-url: "https://discord.com/api/webhooks/..."

# 送信時のBot設定
username: "Hako-Notify"
avatar-url: "https://example.com/icon.png"

活用シーン

運用ログ

サーバーの起動・停止を自動通知し、稼働状況を把握。

緊急連絡

管理者間の連絡をゲーム内からDiscordの緊急用チャンネルへ転送。

ログ記録

重要イベントを記録し、後からDiscord上で検索・確認可能に。
