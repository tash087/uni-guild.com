🤝 運営紹介・モットー (About Us)

✨ ゆにくら！のモットー

「ユニークな創造と温かい交流」

私たち「ゆにくら！」運営チームは、全てのプレイヤーが安心して、自分だけのユニークな建築や大規模なプロジェクトに没頭できる環境を提供することを目指しています。また、協力や助け合いを通じて温かいコミュニティを育むことを大切にしています。

🛡️ 運営方針

  • 長期安定運営: 生活ワールドのリセットは行わず、長期間の建築を保証します。
  • 公平性の追求: 全てのプレイヤーに対し、ルールに基づいて公平な対応を行います。運営メンバーは職務に必要な管理権限を保持しますが、ゲームプレイにおいては私的な利益のためにこれを行使しません。
  • 徹底した荒らし対策: ロールバック機能を駆使し、荒らし行為には迅速かつ厳正に対処します。

鯖主 tash087 からのメッセージ

「ゆにくら！」は、私が理想とする、大人が真剣に遊べるサバイバルワールドを目指して立ち上げました。プレイヤーの皆さんの創造性とアイデアがこのサーバーの全てです。一緒に素晴らしいコミュニティを作り上げていきましょう！

👨‍💻 運営チーム紹介

「ゆにくら！」を支える運営メンバーたち

tash087 (鯖主/Founder)

サーバーの設立者兼全体責任者。技術的な管理（ホスティング、システム構築）、資金調達、最終的な意思決定を担います。コミュニティが活発になるよう、縁の下の力持ちとして活動しています。

nimo_modoki (Cre / PdM / CS / PR)

クリエイティブ部門、プロダクトマネージメント部門、カスタマーサポート部門、広報部門を兼任。サーバーの企画立案から、プレイヤーの皆さんの声を聞いた改善、新しいコンテンツの創造、情報発信まで幅広く担当します。「遊び心を形にするのが大好きです！」

kosasokami (Eng / Cre)

エンジニアリング部門とクリエイティブ部門を兼任。サーバーの技術的な安定性を保ちつつ、イベントやワールドコンテンツの創造にも貢献します。インフラとデザインの両面から、プレイヤーの皆さんの体験を向上させることを目指します。「技術力と創造性を融合させ、ゆにくら！に新しい楽しさをもたらします！」

pam0326 (クリエイティブ部門/Cre)

主にクリエイティブ部門を担当し、ワールドの景観デザインや公共施設の建築、イベント装飾など、視覚的に楽しい空間づくりを担います。「美しいワールド作りと、皆さんの笑顔のために頑張ります！」
