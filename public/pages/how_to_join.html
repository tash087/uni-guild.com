🚀 参加方法 (How to Join)

当サーバーはMinecraft Java Edition のみ対応しています。以下の手順に従って、ゆにくら！サーバーへ接続してください。

1. 動作環境の確認

当サーバーは以下の環境での動作を推奨しています。

  • エディション: Minecraft Java Edition (PC版) のみ
  • 推奨バージョン: 1.21.4
  • 必須: Minecraftの正規ライセンスを所有していること

重要

統合版（Bedrock Edition, PE, Switch, PS4/5など）からは接続できませんのでご注意ください。

2. IPアドレスの確認とコピー

以下のIPアドレスをコピーして、接続準備をしてください。

IP Address: play.uni-guild.com

3. サーバーへの接続手順

  1. Minecraft Java Editionを起動します。
  2. タイトル画面で「マルチプレイ」を選択します。
  3. マルチプレイ画面で「サーバーを追加」を選択します。
  4. 以下の情報を入力します。
    • サーバー名: ゆにくら！ (任意)
    • サーバーアドレス: 2.でコピーした `play.uni-guild.com`
  5. 「完了」をクリックすると、サーバーリストに「ゆにくら！」が追加されます。
  6. リストから「ゆにくら！」を選択し、「サーバーに接続」をクリックすれば参加完了です。

接続がうまくいかない場合は、FAQをご確認いただくか、Discordサーバーのサポートチャンネルにてお問い合わせください。
