💎 建築ランク詳細[箱庭クラフト！]

箱庭クラフトでは、個々の創造性を尊重し、建築の「実績（数）」と「質（技術スコア）」に基づいたランク制度を採用しています。提出作品のクオリティが著しく高い場合、初期ランクを飛ばして上位からスタートできる「飛び級制度」を設けています。

I. 査定ボーナスと特別制度

✨ 傑作（Masterpiece）認定： 提出された1つの作品が「熟練」レベルの質（細部へのこだわり、独自のブロック選定、ストーリー性）を持つ場合、中間のステップを飛ばして直接昇格が可能です。

🖼️ ポートフォリオ評価： 他サーバーやシングルプレイでの実績（画像・動画）を提示し、同等のものを当サーバーで再現できると証明された場合、初期ランクを「中堅」以上に設定します。

II. ランク階級・昇格規定一覧

1. 新米建築家 (Novice Builder)

建築の基礎を学び始めた段階です。

🚀 昇格・認定条件

  • 通常昇格： 小規模な建築物（家1軒など）を1つ完成させる。
  • 飛び級： 設定なし（まずはサーバーの雰囲気に慣れる段階）。

🎁 権限・特典

基本的なワープコマンドの使用権。

2. 中堅建築家 (Journeyman)

建築の「型」ができ始め、装飾にこだわりが見られる段階です。

🚀 昇格・認定条件

  • 通常昇格： 建築物3つ以上。または合計1000ブロック以上の規模の建築。
⚡ 飛び級： 最初の1軒が「内装・外装ともに非常に優れている」と運営に判断された場合、新米を飛ばしてここからスタートします。

🎁 権限・特典

一部便利な建築支援コマンドの解放。

3. 熟練建築家 (Experienced Builder)

街の景観を意識し、地形との調和や独自のスタイルを持ち始めた段階です。

🚀 昇格・認定条件

  • 通常昇格： 中規模以上の建築物5つ以上。かつ、異なるバイオームやスタイルでの建築実績。
⚡ 飛び級（スタート目安）： 「大規模建築（100×100以上）」を1つ完成させ、運営審査で80点以上を獲得した場合、中堅を経ずに昇格。

🎁 権限・特典

恒久ワープ地点の作成権、専用プレフィックスの付与。

4. 筆頭建築家 (Lead Architect)

サーバーの顔となる建築を生み出し、他のプレイヤーの手本となる段階です。

🚀 昇格・認定条件

  • 通常昇格： 熟練ランクとして活動し、公共施設や大型プロジェクトを2つ以上完遂。
⚡ 飛び級： 圧倒的な技術（WorldEdit級の造形、複雑な回路、超細密な内装）が認められ、運営の全会一致で認定された場合。

🎁 権限・特典

建築保護エリアの拡張、WorldEdit系コマンドの一部利用権。

5. 匠 (Master / Grandmaster) - 👑 最高位

サーバーの歴史に名を刻む「遺産」レベルの建築を行うランクです。

🚀 昇格・認定条件

  • 通常昇格： 筆頭建築家として長期間活動し、サーバーを象徴するランドマーク（エリア一帯の都市開発など）を完成させること。
⚠️ 制限： 飛び級は不可。技術だけでなく、サーバー内での信頼と継続的な貢献が必須となります。

🎁 権限・特典

運営会議への助言権、ワープリスト最上位への掲載、専用展示区画の提供。

III. 🔍 審査・査定基準

実力認定は以下の基準でスコア化されます。

評価項目 基準
完成度 (85%以上)未完成（裏側が豆腐、内装が空っぽ等）でないこと。
景観配慮 (80%以上)周囲の建築や地形を壊さず、より良く見せているか。
独自技術既存テンプレの模倣ではなく、自分なりの工夫があるか。
